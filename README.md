# School Management System

A Python backend-based school management system for managing student admissions and records using Excel automation.

## Features

* New Student Admission
* Search Student
* Search By:
  * Student Name
  * Father Name
  * Phone Number
* Update Student Details
* Leave Student Management
* Excel-Based Persistent Storage
* Smart Input Handling

## Technologies Used

* Python
* openpyxl
* Excel
* Git & GitHub

## Project Structure

main.py
admission.py
search.py
update.py
leave.py
students.xlsx



## How To Run

Install required library:
pip install openpyxl


Run project:

python main.py
